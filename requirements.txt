# Python Requirements File

# INSTALLATION INSTRUCTIONS:
# 
# 1. Create a Python virtual environment:
#    python -m venv venv
#    
# 2. Activate the virtual environment:
#    - Windows: venv\Scripts\activate
#    - macOS/Linux: source venv/bin/activate
#    
# 3. Install dependencies from this file:
#    pip install -r requirements.txt
#    
# 4. To deactivate the virtual environment when done:
#    deactivate
#
# NOTE: Some packages may require additional system dependencies:
# - vosk: Requires pre-trained models to be downloaded separately
# - pyaudio: May require portaudio system library
# - sounddevice: May require additional audio drivers
# - pyttsx3: Uses system TTS engines
